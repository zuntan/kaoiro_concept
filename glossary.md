# 用語集 (glossary.md)

プロジェクト「kaoiro」の開発・設計において使用される専門用語、略語、および固有の概念定義。

## 1. コアコンセプト (Core Concepts)

| 用語 | 英語表記 | 定義・解説 |
| :--- | :--- | :--- |
| **カオイロ** | **kaoiro** | 本プロジェクトの名称。顔のジェスチャーを用いたハンズフリー入力支援ツール。 |
| **アバター** | **Avatar** | 認識状態（Idle/Yes/No/Lost）をユーザーに可視化するための顔型グラフィック。SVG形式で描画される。以前は "Hero Icon" と呼称されていたが廃止された。 |
| **ジェスチャーアクション** | **Gesture Action** | ユーザーの顔の動き（Nod, Shake）によってトリガーされる操作イベント。 |
| **プライバシービュー** | **Privacy View** | カメラ映像をそのまま表示せず、ぼかしや線画（ランドマーク描画）加工を施した状態で表示する機能。PiP (Picture in Picture) として実装される。 |
| **ウィンドウガード** | **Window Guard** | 誤入力を防ぐため、特定のウィンドウがアクティブな時のみキー入力を送信する機能。 |

## 2. アーキテクチャ・技術 (Architecture & Tech)

| 用語 | 英語表記 | 定義・解説 |
| :--- | :--- | :--- |
| **デスクトップ版** | **Desktop** | Rust + egui + ort で動作するデスクトップアプリケーション。Windows/macOS/Linux対応。MCPサーバー機能を含む。 |
| **ブラウザ拡張機能版** | **Browser Extension** | Webブラウザ上で動作する拡張機能。WASM + Web Worker を利用して動作する。MCP機能は持たない。 |
| **単一バイナリ** | **Single Binary** | デスクトップ版の配布形態。GUI機能とMCPサーバー機能が1つの実行ファイル（.exe等）に内包されている状態。 |
| **MCP** | **Model Context Protocol** | AIエージェント（Claude, Cursor等）とツールが通信するためのプロトコル。本アプリではジェスチャー状態をエージェントに通知するために使用。 |
| **egui** | **egui** | Rust製の即時モードGUIライブラリ。軽量でクロスプラットフォームなUI構築に使用。 |
| **ort** | **ort** | ONNX Runtime の Rust バインディング。デスクトップ版での高速なAI推論に使用。 |
| **tract** | **tract** | Rust 純製の ONNX 推論エンジン。ブラウザ拡張機能版での推論に使用。 |
| **Vosk** | **Vosk** | オフライン音声認識エンジン。デスクトップ版での音声コマンド機能に使用。 |

## 3. ジェスチャー判定ロジック (Logic)

| 用語 | 英語表記 | 定義・解説 |
| :--- | :--- | :--- |
| **相対位置戦略** | **Relative Position Strategy** | 顔の平行移動（Z軸含む）を無視し、顔面基準プレーンに対する鼻先の相対位置変化から回転成分のみを抽出する判定アルゴリズム。 |
| **動的キャリブレーション** | **Dynamic Calibration** | ユーザーの「真正面」の姿勢を固定値とせず、直近数秒間の平均姿勢をゼロ点として自動更新し続ける仕組み。 |
| **顔アンカー** | **KaoAnchor** | AIモデルによって検出される顔の特徴点（鼻先、目、口など）。一般的呼称は「ランドマーク」だが、本プロジェクトでは基準プレーンやベクトルの基点として「錨（いかり）」の役割を果たすためこの名称を使用する。 |
| **デプスニュートラライゼーション** | **Depth Neutralization** | カメラとの距離（Z軸/深度）による座標値の変動を無効化（中和）し、常に一定のスケールで扱うための比率変換処理。 |
| **インテントスタビライザー** | **Intent Stabilizer** | 判定のON/OFF閾値に決定的な余裕幅を持たせることで、数値のバタつきを抑え、ユーザーの「意思 (Intent)」を安定的に抽出する仕組み。一般的呼称は「ヒステリシス」。 |
